# Jusic Market

가상 멀티마켓 주식 웹게임입니다. 가상의 상장사가 매 거래일마다 생산, 수주, 임상, 업데이트, 원가 협상 같은 업무를 수행하고, 그 성과가 주가와 연관주에 전파됩니다.

## 실행

```bash
npm install
npm run dev
```

개발 서버 기본 주소는 `http://127.0.0.1:5173`입니다.

## 들어간 규칙

- 국내, 미국, 일본 80개 종목
- KOSPI/KOSDAQ/NASDAQ/NYSE/TSE/MOTHERS, 섹터, 연관주 민감도
- 기준가 대비 일일 가격제한폭 `±30%`
- 과열 후 차익실현, 평균회귀, 원자재/환율 비용, 해외장 충격
- 가격대별 호가 단위 반영
- 국내/해외 매도 시 세금, 위탁수수료, 환전 비용 반영
- 일반 계좌와 ISA 계좌 분리
- ISA 연간 납입한도와 비과세/저율과세 모델
- 뉴스사 3곳의 관점, 오보 리스크, 뉴스사 주가 영향
- 악재 누적에 따른 관리종목, 비상상황, 구제, 드문 상장폐지
- 신규 상장, 세금/환율 계산기, 알림, 스크리너, 미션, 캘린더
- 거래일 진행, 회사별 업무 성과, 뉴스 테이프, 섹터 열지도

## 광고 설정

광고 슬롯은 상단, 주문 패널, 뉴스 패널에 준비되어 있습니다. 실제 광고를 띄우려면 광고 네트워크 승인 후 GitHub 저장소의 `Settings > Secrets and variables > Actions > Variables`에 아래 값을 추가하세요.

```bash
VITE_ADSENSE_CLIENT=ca-pub-3303941146778727
VITE_ADSENSE_SLOT_TOP=2342953263
VITE_ADSENSE_SLOT_SIDE=...
VITE_ADSENSE_SLOT_NEWS=...
```

AdSense client id와 상단 광고 슬롯 ID는 기본값으로 코드에 들어가 있습니다. 나머지 슬롯 ID가 없으면 자동 광고 스크립트만 연결되고, 수동 광고 위치에는 플레이스홀더가 보입니다.

실제 제도는 게임용으로 단순화되어 있습니다.
